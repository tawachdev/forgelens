# Security Policy

## Supported versions

ForgeLens is currently pre-1.0. Security fixes are applied on the latest published minor release.

## Report a vulnerability

Please do not open public GitHub issues for security reports.

Report privately using one of these options:

- GitHub Security Advisories (preferred)

Include:

- What you found
- Impact
- Steps to reproduce
- Suggested fix (if known)

## Response goals

- Acknowledge report within 72 hours
- Confirm severity and next steps
- Ship a fix as fast as possible based on impact

## Scope notes

ForgeLens is a static-analysis CLI. It does not execute scanned app code and must not expose secret env values.
